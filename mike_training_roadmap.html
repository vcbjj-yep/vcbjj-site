Regression Drill from Americana — Building Tactical Reading

Working backwards to find where the reading breaks down

Why Regression, Not More Technique
You came in with a clear self-diagnosis: you don't move your hips enough, your movements feel textbook, you cannot identify how and when to move efficiently. That is not a technique problem — it is a reading problem. You move through a memorised sequence rather than reading which door is open. Adding more techniques would make this worse, not better. The regression drill addresses the actual cause.
The Regression Drill — How It Works
Start at the end: chest-to-chest pin with the Americana already applied. Remove one control function (a grip, a post, a position). You fight to re-earn it. Once re-earned, that function is removed again plus one more. Each iteration you are starting from further back. The goal is not to complete the Americana — it is to read, from each reduced position, which function to pursue next. That is the door-reading skill.
The Regression Pathway
Americana applied → remove one grip → re-earn → remove two controls → re-earn → ... → remove all controls (just chest-to-chest) → re-earn → working from hip line entry → working from open guard or standing. Same funnel, entered from the beginning. By the time you have worked the whole pathway, you have physically experienced every decision point in the chain — not read about it.
Connection to Session #35
Session #35 clarified what a funnel is — attacks that change as the position changes, not technique adjustments within the same attack. Session #36 is the drill that builds the reading ability the funnel requires. You cannot run the funnel if you cannot read the open door. The regression drill builds exactly that — by forcing you to identify, under resistance, what is available from each reduced state.
Follow-up required: Hip awareness and movement efficiency — entries are still telegraphed and sequential rather than positionally reactive. The regression drill is the ongoing prescription. Each session: one more step back in the regression pathway. No new techniques needed yet.
